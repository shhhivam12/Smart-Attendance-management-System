Smart Attendance Management System Using Face Recognition

Welcome to Graphic Era's Smart Attendance Management System!


Revolutionizing the way we track attendance with my web-app solution designed specifically for Graphic Era. Smart Attendance Management System leverages the latest advancements in technology of computer vision to streamline the attendance-taking process, making it more efficient, accurate, and hassle-free for both students and faculty.

How does it works?

The core concepts used in SAMS is facial recognition which is implemented through popular computer vision library - OpenCv. In this project i am using python to implement OpenCv in web with the help of html, css and javascript. The backend of this project is handled by Flask. This project aims at reducing the time overrheads inccured through traditional ways of taking attendance in schools and colleges by automating the process which matches faces of students through computer vision and provide the faculty with the roll numbers of the students at the end of the class
